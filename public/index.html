VidCrop

Crop and trim videos, use FFmpeg to create GIF or MP4 snippets.

VidCrop lets you crop a video and trim it to a time segment. It does not in itself generate anything or process the files in any way, but is an interface to put together the ffmpeg commands you need to generate GIFs or video clips for all your memeing needs.

You'll need FFmpeg installed on your computer, and to run the generated commands in the same folder as your video file.

VidCrop is a project by MichD. You can find it on GitHub: michd/VidCrop.
